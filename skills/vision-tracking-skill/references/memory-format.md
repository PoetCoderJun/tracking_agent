# Memory Format

The tracking memory should be a single short paragraph.

## Rules

- Do not split the memory into sections or bullet lists.
- Write one short paragraph in natural language.
- Keep memory compact and optimized for the next search turn.
- Prefer rewriting the paragraph over appending logs.
- First describe the target's traits in as much useful detail as possible.
- Then state how to distinguish the target from nearby people.
- Distinguish observations from hypotheses in wording.

## Example skeleton

```md
# Tracking Memory

短发，白色上衣，浅色短裤，白袜深色鞋，戴眼镜，体型偏瘦。和周围人区分时优先看眼镜、裤长、鞋子配色和体型。
```
